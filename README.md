### 설치 및 실행

1. 저장소 복제 (Clone Repository)

```zsh
$ git clone https://github.com/LIKELION-FE-LAB9/Market-Karly.git
```

2. 종속성 설치 (Install Dependencies)

```zsh
$ npm install
```

3. 애플리케이션, JSON 서버 실행 (Run Application & JSON Server)

```zsh
$ npm run all
```
